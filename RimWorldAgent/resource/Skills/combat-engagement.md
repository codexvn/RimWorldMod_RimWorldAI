# 野战接敌

## 核心循环

```
①观察 → ②选位 → ③执行 → ④推进(600-1200) → 回到①
```

战斗不是一轮操作然后等结果。每次推进后必须重新观察、重新决策。

## 阶段 0：装备（非征召状态）

### 流程（必须严格遵守）

```
equip_pawn / force_dress → advance_tick 等待捡完 → get_defense_status 确认 → draft_pawn
```

**禁止** `equip_pawn` 后立刻 `draft_pawn`。征召状态殖民者不去捡装备。

### 装备策略

- **用 `get_thing_stats` 看武器数值后决定分配**——精度/穿甲/DPS 直接可见
- **武器最佳距离 = 射程 × 80%**（`shooting_position_grid` 的评分以此为准）
  栓动步枪 37 格 → 最优 29 格；左轮手枪 26 格 → 最优 21 格
- **精度曲线决定武器用法**：左轮手枪贴脸 80% 降远 40%，应拉近；栓动步枪中距离 90%，保持中距离
- **护甲优先级**：近战 > 远程 > 非战斗。近战吸引火力，必须穿防弹三件套
- **全员至少一件护甲**：无甲 pawn 被霰弹枪一枪可能直接倒地（实战案例：Pheanox 无甲 34% vs Rafael 有甲 100%）

### 征召后检查

`draft_pawn` 返回每个人：移速、护甲（利刃/钝击/热能）、开火状态、武器完整属性。据此确认配置无误。

## 阶段 ①：观察

```
find_enemies(show_movement=true) + get_colonists
```

- `show_movement=true` 显示：**攻击目标**、**ETA 秒数**、移动路径
- ETA > 10s → 有时间选位、找掩体
- ETA < 5s → 紧急布防，直接用 `hold_combat_position` 就近防守
- **留意敌人武器类型**：霰弹枪近距离致命、步枪中距离危险、近战需要堵门

## 阶段 ②：选位

**优先找建筑掩体**。RimWorld 中墙角/建筑边缘是最强掩体（墙角探身射击 = 高达 75%+ 掩护效率），远优于沙袋（40%）和树木（25%）。

### 选位流程（按优先级）

1. **`get_tile_grid` 查看当前区域**，找建筑/墙体/废墟的拐角和边缘。墙角的相邻格如果有射界 = 最佳射击位。
2. **`shooting_position_grid`** 扫描掩体评分，Top 5 中优先选靠墙位置。
3. 如果扫不到位（全平地/没打到目标），**人工目视**：`get_tile_grid` 看敌人方向哪里有墙、石头、山体边缘，手动算坐标站位。
4. **绝不放弃找掩体**。扫不到就扩大范围、换 chunk、缩短到敌人距离再扫。平地对射 = 自杀。

### 建筑掩体优先级

| 掩体 | 效率 | 站位 |
|------|------|------|
| 墙拐角相邻格 | 75%+ | 站墙角两侧，探身射击 |
| 建筑门口两侧 | 50-75% | 靠墙站，门框遮一半 |
| 沙袋/路障 | 40-57% | 正后方 |
| 树干/石头 | 25-50% | 正后方 |
| 无掩体 | 0% | 禁止——霰弹一枪倒地 |

- 墙角相邻格只会被部分方向打中。敌人从正面来 → 墙遮住暴露面 → 远程安全和掩体兼顾
- 多个 pawn 不要挤同一个墙角——射线互相遮挡
- 近战不选位：近战选门口的**门外 1 格**堵口，掩体对它没意义

### 无建筑时的替代

平地上找天然掩体：树木（尤其成簇）、石块、山坡边缘。都没有 → 主动向敌人方向走，缩短距离直到 `shooting_position_grid` 能打到为止。

## 阶段 ③：执行

| 命令 | 用途 |
|------|------|
| `hold_combat_position(role="ranged")` | 远程守位+自动开火，**防守首选** |
| `hold_combat_position(role="melee")` | 近战堵口守位，站门外 1 格 |
| `force_attack(attack_mode="ranged")` | 远程集火特定目标，校验射程/视线/距离 |
| `force_attack(attack_mode="melee")` | 近战冲锋追杀，AI 自动路径→贴脸 |

`force_attack` 返回"太近"→换 `melee`；返回"无法击中目标"→无视线，用 `shooting_position_grid` 换位。

## 阶段 ④：推进

| 情况 | ticks | 原因 |
|------|-------|------|
| 贴脸肉搏 | 600-900 | 多轮近战交换可见 |
| 远程对射 | 600-1200 | 几轮射击见分晓 |
| 敌人在路上 | 1200-2500 | ETA 判断，留给部署时间 |

**禁止** <500 ticks——看不清因果。每次推进后立即 `find_enemies` + `get_colonists`。

## 近战缠斗

**核心价值**：陷入近战的敌人**无法开枪**、移动减速。近战贴脸 = 废除对方远程。

- 近战必须穿最高护甲（防弹三件套），因为要硬吃伤害
- 近战站窄口/门口，不是站空地——空地敌人直接绕过
- 堵门公式：站门**外**1 格，让后排远程越过肩膀安全输出
- 敌人有霰弹枪时，**让近战主动冲上去贴脸**——近战缠住后霰弹枪废了

## 撤退

- 战斗人员在野外**撤退无用**——敌人追着砍。只能咬牙打完或进门关门
- 非战斗人员可以撤：`move_pawn` 到安全区

## 常见陷阱

1. **征召后装备捡不到** → 先捡完再征召
2. **近战站平地** → 敌人绕过打后排
3. **不找掩体平地对射** → 用 `shooting_position_grid`
4. **不移速匹配推进量** → ETA 5s 的虫子用 600 ticks 检查，不要等 3000
5. **不用 hold_combat_position 防守** → 守位来一个打一个，比 `force_attack` 更省心
6. **对霰弹枪无甲对射** → 一枪穿肾穿肺。近战冲脸或保持距离+掩体
7. **无医药无床也先止血** → `tend_now` 无药治疗也能救流血
