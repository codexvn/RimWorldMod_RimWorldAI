# 动物驯养与管理指南

## 动物分类
### 驮兽 (Pack Animals)
- **Muffalo**：最佳综合选择——负重 84kg、毛产、肉
- **单峰骆驼 (Dromedary)**：沙漠专用，负重/奶产
- **马 (Horse)**：最快驮兽，远行队加速首选
- **驴 (Donkey)**：廉价替代品

### 毛产动物 (Wool)
- **Muffalo**：muffalo wool（高保暖、高防护）
- **羊驼 (Alpaca)**：Alpaca wool（中保暖）
- **绵羊 (Sheep)**：Sheep wool（基础保暖）
- **巨懒 (Megasloth)**：重型毛（最高防护值，极慢产）

### 奶产动物 (Milk)
- **奶牛 (Cow)**：产量最高
- **牦牛 (Yak)**：适应寒冷
- **单峰骆驼**：适应炎热
- **山羊 (Goat)**：廉价产能

### 蛋产动物 (Eggs)
- **鸡 (Chicken)**：最快繁殖，蛋+肉
- **鸭 (Duck)**：替代鸡
- **火鸡 (Turkey)**：较大肉量

### 战斗伙伴 (Combat)
- **熊 (Grizzly/Polar Bear)**：高伤害高坦度，但食量大
- **狼/ Husky / Labrador**：中等伤害低食量，哈士奇可驮运
- **Warg**：纯肉食，伤害高但饲养成本高
- **Thrumbo**：终极战斗伙伴但极难驯服，食量极大
- **巨蜘蛛 (Megaspider)**：不可驯服但可养在洞穴

## 驯服策略
- **驯服等级 (Handling skill)**：每种动物有最低驯服要求
- **野生性 (Wildness)**：越低越容易驯服
- **失败风险**：高野生性动物驯服失败可能激怒（攻击）
- **优先驯服**：驮兽/毛产 → 食物链 → 战斗伙伴

## 饲养管理
- **食物消耗**：食草 > 食肉（成本更低）
- **围栏/睡眠点**：室内睡眠点 + 室外放牧区
- **屠宰控制**：用 Auto-slaughter 自动管理种群上限
- **训练**：Guard/Obedience/Rescue/Haul — 需要持续训练维护
- **区域限制**：室内动物（宠物）vs 室外动物（牲畜）

## 常见问题
- 过度繁殖 → 吃光地图植物 → 其他动物饿死
- 肉食动物食物链 → Warg 吃干肉/kibble 太贵
- 瘟疫（Animal plague）→ 隔离 + 治疗或提前屠宰
- 复仇（Revenge）→ 驯服失败后全组征召
