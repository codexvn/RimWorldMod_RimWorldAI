# 灭火与防火指南

## 火灾优先级
1. **电力设备**（电池/发电设施 → 爆炸风险）
2. **存储区**（物资损失）
3. **居住区/工作间**（家具、生产设施）
4. **种植区**（作物损失可恢复）
5. **野外**（通常不需要手动扑灭，等下雨）

## 灭火操作
- 征召全体殖民者 → 手动右键火焰 → 优先灭火
- 征召的人会自动参与周围灭火，但手动指挥更快
- `Firefighter` 工作类型设为 1 的殖民者自动参与灭火
- 泡沫灭火器（Firefoam popper）可立即扑灭周围火焰

## 防火措施
- **防火带**：基地外围 3 格宽混凝土地板隔离带（非易燃）
- **泡沫灭火器**：关键设备旁放置（电池、冷库、存储区）
- **建筑材料**：石材墙 + 非燃地板（远离木墙钢墙）
- **电缆埋设**：石墙内走电缆，减少短路起火
- **分区隔离**：存储区之间留防火间隔
- **屋顶管理**：室内火灾 → 拆除屋顶释放热量 → 温度骤降灭火

## 特殊火灾类型
- **电池短路 (Zzzt...)**：立即泡沫弹覆盖电池区，切断电路
- **爆炸火灾**：泡沫+灭火双线并行
- **Boomalope/Boomrat 爆炸**：不要在它们旁边枪战，远程射杀后泡沫覆盖
- **山火/闪电**：观察风向，防火带外等雨或手动扑灭关键区域

## 火灾预防检查清单
- 电池不要密集堆放
- 存储区泡沫灭火器覆盖范围
- 基地外围防火带宽度 ≥ 2 格
- 关键区域石材建造
- 雨天进行爆炸性操作
