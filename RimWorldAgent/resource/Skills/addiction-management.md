# 成瘾与药物风险管理

## 适用场景
- 殖民者出现药物成瘾、戒断、过量或频繁自行服药
- 殖民者有化学兴趣、化学迷恋等容易滥用药物的特性
- 地图上有清醒药、槟榔茶、烟叶、啤酒、醒神剂、魔鬼素等成瘾品
- 需要为战斗、医疗或心情危机决定是否临时用药

## 总原则
- 先保命，再戒断；急性战斗、感染、极端心情崩溃风险优先处理
- 不要为了短期效率随意使用高成瘾风险药物
- 成瘾管理的核心是减少可达性、稳定心情、安排休养和监控戒断进度
- 不要重复让同一殖民者服用可能加重成瘾的药物，除非是在处理魔鬼素依赖或明确的紧急情况

## 成瘾如何获得
- 成瘾通常来自反复服用带成瘾风险的药物或娱乐品；部分药物单次使用也可能触发成瘾
- 成瘾风险与药物类型、使用频率、个体状态和游戏设定相关；不要把“偶尔使用安全”当成绝对规则
- `化学兴趣` 和 `化学迷恋` 会显著提高主动服药风险，这类殖民者更容易自行寻找成瘾品
- 药物放在公共仓库、餐厅附近、通道边或未禁止状态，会增加殖民者接触和滥用机会
- 心情低、压力大、长期疼痛、睡眠不足时，殖民者更可能通过药物缓解状态
- 已有耐受不等于成瘾，但说明使用频率已经偏高，应立即降低使用频率并复查健康
- 魔鬼素依赖是特殊高危状态，一旦依赖就不能按普通戒断处理

## 成瘾控制手段
- **控制来源**：用 `search_map` 或 `get_resources` 找出地图上的成瘾品，散落药物先用 `forbid_item` 禁止
- **控制库存**：建立独立药物储藏区，不要与食物、常用医药或主仓库混放
- **控制接触**：化学兴趣/化学迷恋者不要负责搬运成瘾品，不要长时间靠近药物仓库工作
- **控制服用**：只有明确需要时才用 `ingest_item` 指定殖民者和具体物品，不要泛化为全员策略
- **控制节奏**：每次用药后记录原因，之后用 `advance_tick` 推进并复查健康、心情、耐受和成瘾状态
- **控制替代方案**：优先用环境、睡眠、娱乐、食物质量、疼痛处理和轻工作改善状态，不把药物当作长期心情方案
- **控制高危对象**：对已成瘾或戒断者降低工作负担，必要时 `force_bed_rest`，避免战斗、远行、关键手术和重体力工作
- **控制魔鬼素**：非依赖者原则上不使用；依赖者必须优先保证供应，不能简单禁止或断供

## 成瘾品的获取
- **地图现有物资**：用 `search_map(keyword="药物", category="item")`、具体 defName 或 `get_resources` 查找散落药物、战利品、商队掉落和遗迹物资
- **贸易购买**：用 `list_faction_traders` 查看可贸易派系，再用 `trade_execute` 预览不同商船库存；只在有明确用途时购买成瘾品
- **任务奖励**：接受任务前检查奖励，清醒药、醒神剂、魔鬼素等高风险药物只在殖民地有管控能力时接收
- **生产制造**：如果已解锁相关研究和工作台，可通过生产单据制造药物；制造前先确认是否真的需要长期库存
- **种植和加工**：部分成瘾品来自作物或加工链，例如精神叶相关产品；种植前要考虑库存隔离和使用政策
- **敌人和访客遗留**：战斗后检查战场掉落，先救治和清理威胁，再决定是否允许、搬运或禁止成瘾品
- **特殊依赖补给**：魔鬼素依赖者需要稳定补给，获取魔鬼素是维持生命的优先事项，而不是普通娱乐或效率提升

获取前必须判断：
- 是否有人已经成瘾或依赖，是否存在断供风险
- 是否有化学兴趣/化学迷恋者会滥用库存
- 是否有独立储藏区和禁止/允许管理手段
- 是否有足够白银、贸易机会或生产能力维持后续供应
- 该药物是救命、战斗应急、医疗应急，还是单纯提升效率

不要主动获取：
- 没有依赖者、没有应急需求、只是为了短期效率的魔鬼素
- 当前殖民地心情崩溃频发且没有库存隔离能力时的大量成瘾品
- 白银不足、食物/医药/防御短缺时的非必要娱乐药物
- 无法安全储存且容易被化学兴趣/化学迷恋者接触的药物

## 第一步：确认问题
1. 用 `get_colonists` 查看殖民者特性、能力和当前工作。
2. 用 `get_colonist_health` 查看成瘾、戒断、耐受、过量、意识和移动能力。
3. 用 `get_colonist_needs` 查看心情、娱乐、睡眠和即将崩溃的风险。
4. 用 `search_map` 或 `get_resources` 查找地图上的药物和成瘾品库存。

重点判断：
- 是已经成瘾，还是只有耐受/风险
- 是普通成瘾戒断，还是魔鬼素依赖
- 是否存在精神崩溃、重伤、感染、饥饿、低睡眠等叠加风险
- 药物是否散落在殖民者可随手拿到的位置

## 普通成瘾处理
- 停止继续服用对应药物，让戒断进度自然下降
- 将成瘾品移动到受控储藏区，或用 `forbid_item` 禁止散落药物
- 设置患者 `Patient = 1`、`PatientBedRest = 1`，必要时用 `force_bed_rest` 让其休养
- 降低该殖民者工作负担，避免安排战斗、长途搬运、高风险手术或关键生产
- 提升心情缓冲：精致食物、干净卧室、娱乐时间、轻工作、避免尸体和脏乱环境
- 用 `advance_tick(hours=0.5~2)` 推进一段时间后复查健康和心情，不要连续重复下同一命令

## 魔鬼素依赖
- 魔鬼素依赖不能按普通戒断处理；断供会持续恶化并最终致命
- 若已有魔鬼素依赖，优先保证稳定供应，而不是立即禁止所有魔鬼素
- 统计库存和服用间隔，优先通过贸易、任务奖励或地图搜索补充
- 库存不足时将依赖者列为最高风险对象，避免派出远征或参与高伤亡战斗
- 只有在明确有长期供应链时，才考虑让非依赖者使用魔鬼素

## 药物库存隔离
- 成瘾品不要混在公共食物区或常用仓库里
- 建议建立独立药物储藏区，靠近医院但远离餐厅和主要通道
- 对散落的高风险药物使用 `forbid_item`，需要时再精确 `allow_item`
- 如果必须使用药物，用 `ingest_item` 指定具体殖民者和具体物品，避免不受控服用
- 对清醒药、醒神剂、魔鬼素等高风险药物，优先保留给明确战斗、医疗或生命风险场景

## 化学兴趣/化学迷恋
- 有化学兴趣或化学迷恋的殖民者更容易主动滥用药物
- 不要让这类殖民者负责搬运或长期接触成瘾品库存
- 优先给他们稳定娱乐、睡眠和心情来源，降低主动嗑药诱因
- 成瘾品散落时先禁止或搬离，再推进时间观察

## 安全用药原则
| 药物类型 | 建议 |
|----------|------|
| 盘诺西林 | 主要用于预防疾病，按周期使用，通常不是成瘾管理重点 |
| 精神茶 | 风险较低，但仍避免频繁连续使用 |
| 啤酒/烟叶 | 可缓解心情，但会降低效率或带来成瘾风险，不作为长期心情方案 |
| 清醒药/醒神剂 | 高风险，只用于紧急生产、医疗、战斗恢复等短期场景 |
| 魔鬼素 | 极高风险；非依赖者原则上不使用，依赖者必须保证供应 |

## 复查节奏
- 急性风险：每 0.5~1 游戏小时复查健康和心情
- 稳定戒断：每 2~4 游戏小时复查一次
- 戒断期间如果心情接近崩溃阈值，先处理心情，不要继续硬推时间
- 戒断完成或风险下降后，用 `task_update` 标记相关任务完成

## 禁忌
- 不要把魔鬼素依赖当作普通成瘾强行断供
- 不要在没有复查健康状态时连续使用 `ingest_item`
- 不要让低心情、重伤、感染、饥饿的成瘾者继续承担关键任务
- 不要把成瘾品和食物、常用药品混放
- 不要只靠药物解决心情问题，应优先改善环境、睡眠、娱乐和食物质量
