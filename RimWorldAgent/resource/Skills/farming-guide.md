# 种植与作物选择指南

## 肥力系统
- **肥力敏感度**：作物对土壤肥沃度的依赖程度
  - 高敏感（100%）：水稻、玉米——肥沃地高产，贫瘠地大减
  - 低敏感（60%）：土豆——贫瘠/砾石首选
  - 无敏感（0%）：干草、纤维作物、治疗草——不看肥力
- **土壤肥力**：肥沃土壤(140%) > 普通土壤(100%) > 砾石(70%) > 沙地(10%)
- **水栽培**：固定 280% 肥力，无视外面天气和季节

## 主要作物选择

| 作物 | 生长天数 | 肥力敏感 | 产量 | 推荐场景 |
|------|---------|---------|------|---------|
| 水稻 | ~3 | 100% | 低 | 肥沃地，水栽培 |
| 玉米 | 11.3 | 100% | 高 | 大户种植（产量/工时最佳） |
| 土豆 | ~5 | 60% | 中 | 贫瘠地/砾石 |
| 草莓 | ~4 | 60% | 中 | 可生吃+无烹饪心情惩罚 |
| 干草 (Hay) | ~5 | 0% | 高 | 饲料，贫瘠地随便种 |
| 棉花 | ~7 | 0% | - | 纺织原料，不需要肥力 |
| 治疗草 (Healroot) | ~8 | 0% | - | 基础医药，不需要肥力 |
| 烟叶 (Smokeleaf) | ~5 | 0% | - | 娱乐/贸易 |
| 精神叶 (Psychoid) | ~7 | 0% | - | 精神茶/薄片/仿生体 |

## 种植策略
- **第一季**：水稻保吃饭 + 棉花保穿衣 + 治疗草保医疗
- **中后期**：大部分转玉米（省工时），留少量水稻应急
- **贫瘠地**：土豆覆盖
- **水栽培**：只种水稻（省电 + 最高产出），配合太阳灯 24h

## 水栽培与温室
- **太阳灯**：覆盖范围 100 格（圆形）→ 24 个水栽培盆
- **电力**：太阳灯 2900W（日间），夜间关灯作物不生长
- **温室**：肥沃土壤 + 太阳灯 + 加热器（冬季） → 全年种植
- **水栽培禁用**：不能在室外，只能在有屋顶的封闭房间
- **太阳灯+水栽培盆** = 3.5 天产一批水稻 ≈ 人均需 4 个盆

## 季节管理
- **生长季** (Growing Season)：查看世界地图的季节长度
- **冬季储备**：生长季最后一个月停止收割→全转种植（确保生长期）
- **冷库容量**：冬季前检查冷库食物存量和容量

## 常见问题
- **枯萎病 (Blight)**：立即标记→砍伐→全地块清除→重栽
- **冷潮 (Cold Snap)**：户外作物可能全死 → 收割一切即使未熟
- **热浪 (Heat Wave)**：户外干旱风险，室内水栽培安全
- **日光不足 (Eclipse)**：太阳灯关闭 → 作物生长暂停
