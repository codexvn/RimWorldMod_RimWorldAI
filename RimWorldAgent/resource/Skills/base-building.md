# 基地建造规划指南

## 基地模板（强制优先使用）

建造多房间基地时，**必须优先使用 apply_base_template**，严禁自行计算坐标。LLM 的空间数学不可靠，手工算坐标必然导致房间错位。

### 工作流
1. `list_base_templates` — 查看可用模板及描述
2. 根据殖民地需求和地形选择合适的模板
3. `apply_base_template(template_name, center_x, center_y)` — 获取所有房间精确坐标
4. `designate_room` — 按返回坐标逐一建造
5. 根据实际需要微调（如跳过某个房间、调整单间尺寸）

### 模板选择
- 初期/独立建筑 → `single_room`（单房间，外径/墙体占地 7~13；标准间 external_size=13）
- 中期综合基地 → `nine_grid`（九宫格 3×3，紧凑无走廊）或 `nine_grid_walled`（带外围防御墙）
- 殖民者居住区 → `bedroom_row`（卧室排，5间×5x5 内径）

### 禁止事项
- **严禁**自行计算坐标后调 `designate_room`（房间必然错位）
- **严禁**用 `designate_build` 逐格造墙（效率低且更易出错）
- **严禁**在无 `get_structure_layout` 确认空地的情况下直接建造

### 规划草图（推荐工作流）

建造前先用规划画板画草图，确认布局无误后再正式建造：
1. `plan_add` 画出各房间区域（不同功能区用不同颜色）
2. `plan_list` 查看整体布局，确认房间不重叠、尺寸合理
3. 需要对照地形时 `get_tile_grid` 查看对应区域
4. 确认无误后用 `designate_room` 正式建造
5. 不需要时 `plan_remove` 清理标记

## 标准房间尺寸
- **13x13 (外径/墙体占地)**: 标准万能房间，内部空间 11x11。用 `apply_base_template(..., external_size=13)` 或 `designate_room`。
- **11x11 (外径/墙体占地)**: 紧凑型房间，内部空间 9x9，适合卧室、研究室
- **5x5 或 5x6**: 标准卧室尺寸
- **7x7**: 小型工作间（单一工作台 + 存储）

### 坐标计算示例
**单个房间**：13x13 外径 = 墙体占 13 格。闭区间公式 `end - pos + 1 = size`，验证 `22 - 10 + 1 = 13` → `end = pos + (size - 1)`。

```
起点 (10,0) → 终点 (22,12) → 墙体: x∈[10,22] z∈[0,12]
```

墙体外缘在 x=10、x=22、z=0、z=12 四边，内部空间 11x11。

**两间相邻房间共享墙壁**：Room B 的起点 = Room A 的终点。

```
Room A: (10,0)-(22,12)   — 东墙在 x=22
Room B: (22,0)-(34,12)   — 西墙共用 x=22，B 起点 = A 终点（不是 +1）
```

**严禁**：相邻房间之间留空 1 格（B.pos = A.end + 1）——会浪费材料且堵住 Room A 的入口。

## 柱子与屋顶支撑
- 无支撑屋顶最大跨度 6 格
- 13x13 外径房间内部为 11x11，通常可由四周墙体支撑屋顶；大于 13x13 外径时再考虑加柱子
- 大型空间需要额外的柱子或墙壁

## 功能分区建议

### 默认布局（13x13 外径标准间）
```
┌─────────────────────────────────────────────────┐
│ 13x13 外径 (11x11 内部) x 6 间                  │
│                                                 │
│  冷库+厨房   │  存储区(室内) │  工坊+生产  │
│  Freezer+   │   Stockpile  │  Workshop   │
│  Kitchen    │              │             │
│             │              │             │
│─────────────┼──────────────┼─────────────│
│  研究+医疗  │   餐厅+娱乐   │  居住区     │
│  Lab+Hosp  │   Dining+Rec │  Bedrooms   │
│             │              │   5x5×N     │
│             │              │             │
│─────────────┼──────────────┼─────────────│
│  囚室+仓库  │   种植区/空   │  防御阵地   │
│  Prison+   │   预留地     │  入口+Kill  │
│  Storage   │              │  box 方向   │
└─────────────────────────────────────────────────┘
```

**关键原则**：
- **存储区必须在室内** — 物品露天会腐坏/耐久降低。所有资源类存储区（default/food/raw_resources/manufactured/weapons/apparel）必须放在有屋顶的封闭房间内
- **冷库独立于居住区**，紧邻厨房，入口做双门气闸防温度泄漏
- **厨房区** = 炉灶 + 屠宰台 + 冷库相邻。屠宰台远离炉灶防止交叉污染（食物中毒风险按 Cleanliness 计算）
- **工坊**紧邻室内存储区，减少原材料搬运距离。锻造台+裁缝台+机械加工台放一间
- **研究+医疗**需要无菌地板（+0.6 Cleanliness），可以合用一个无菌房间降低成本
- **餐厅+娱乐**合并到一间（Impressive 叠加），靠近居住区，早晨醒来到达时间短
- **卧室在基地边缘**（非中心），白天不经过不会被打扰。走道供暖+通风口连接卧室
- **医院和囚室在基地中心** — 需要频繁访问，囚犯越狱时更易拦截
- **防御入口朝向预设敌人路径**，外墙 2 格厚花岗岩（HP 510，爆炸阈值以上）

1. **生产区**: 锻造台 + 裁缝台 + 机械加工台 + 存储区紧邻（原材料直达生产台）
2. **厨房区**: 炉灶 + 屠宰台，紧邻冷库双门气闸，屠宰台和炉灶分开放
3. **研究区**: 研究台 + 精密装配台 + 无菌地板。可和医院合用无菌地板省钱
4. **医疗区**: 医疗床 + 生命体征监测仪 + 无菌地板，靠近药品库存
5. **居住区**: 卧室集群，每间 5x5 内径，从走道供暖/通风，基地边缘位置
6. **存储区**: 冷库靠近厨房，普通仓库靠近生产区

## 关键房间文字地图

### 冷库+厨房（双门气闸）
```
       室外
        │
     ┌──门──┐    ← 外层门（气闸入口）
     │ 1格  │    ← 气闸缓冲区
     ├──门──┤    ← 内层门（气闸出口）
     │      │
     │ 炉灶 │ ← 炉灶紧邻冷库门，缩短搬运
     │ Stove│
     │      │
     ├──────┤
     │ 屠宰台│ ← 屠宰台远离炉灶（防交叉污染）
     │Butcher│
     │Table  │
     └──门──┘
```

- 冷库目标温度 **-5°C 以下**，制冷机蓝色制冷端朝内、红色制热端朝室外或无屋顶区域
- 气闸：双层门中间留 1 格空间，防止开关门时温度泄漏
- 炉灶放在冷库门旁 1-2 格，厨师拿食材→烹饪→放回，走路距离最短
- 屠宰台和炉灶至少隔 3 格（屠宰产生污渍降低 Cleanliness → 食物中毒风险）
- 冷库内只放 food 存储区（食材+肉类+饭菜），不放其他杂物

### 单间 5x5 卧室
```
┌─┬─┬─┬─┬─┐
│墙墙墙墙墙│  ← 走道（供暖总管）
├─┼─┼─┼─┼─┤
│墙│床│床│柜│墙│  ← 双人床+床头柜+梳妆台
├─┤  │  │  │─┤
│通│  灯  │墙│  ← 通风口连接走道供暖
│风│  │  │  │─┤
│口│床│床│植│墙│  ← 花盆(美观+1)
├─┼─┼─┼─┼─┤
│墙墙│门│墙墙│  ← 门朝走道
└───┴─┴───┘
  走道方向
```

- 5x5 **内径**（墙体占地 7x7），放双人床+床头柜+梳妆台+花盆
- 通风口开在走道侧墙壁，走道统一供暖/制冷
- 卧室不放生产台（影响房间评级），纯睡眠用
- 初期可临时放研究台/灶台（"卧室+工坊"），后期腾空升级为纯卧室

### 工坊+存储区
```
┌─────────────────────┐
│   室内存储区         │  ← raw_resources + manufactured
│   (Stockpile)       │     原材料+成品放在一起
├──────┬──────┬───────┤
│ 裁缝台│ 锻造台│机械加工│  ← 三台紧邻存储区门
│Tailor│Smithy│Machining│
│      │      │       │
│  ┌───┴──────┴───┐   │
│  │   工具柜×2    │   │  ← 工具柜加速生产（放在工作台旁）
│  └──────────────┘   │
└─────────────────────┘
```

- 存储区必须在**同一间或紧邻**工坊，原材料到生产台走路 ≤5 格
- 锻造台+裁缝台+机械加工台共享工具柜（工具柜效果叠加 ×2）
- 成品区（武器/衣物）可以放在工坊后方的独立存储区

### 医院
```
┌─────────────────┐
│  药品柜 生命体征 │  ← 药品柜+监测仪放中间，覆盖全部床位
│  监测仪         │
│  ┌──┬──┬──┬──┐  │
│  │床│床│床│床│  │  ← 医疗床（无菌地板全覆盖）
│  └──┴──┴──┴──┘  │
│                  │
│   ┌──────────┐   │
│   │奴隶/囚犯床│   │  ← 备用/手术台/囚犯治疗
│   └──────────┘   │
└─────────────────┘
   无菌地板全覆盖
```

- 无菌地板 +0.6 Cleanliness，降低感染率+提升手术成功率+提升治疗质量
- 生命体征监测仪覆盖范围 4 格，放在中间可覆盖多床
- 研究台可以和医院放同一间合用无菌地板（Cleaning room 概念）
- 初期至少 2 个医疗床，中后期 4-6 个

### 防御入口（标准 Killbox 入口）
```
敌人来袭方向 →
          ↓
    ╔══════════╗
    ║  开阔区   ║  ← 15-25 格露天，无掩体（不放柱子！）
    ║  (露天)   ║
    ╚══╤══╤══╤╝
    ┌──┴──┴──┴──┐
    │ 沙沙沙沙沙 │  ← 6 个射手位：沙袋掩体（55%掩护）
    │ 远远远远远 │
    │ 远远远远远 │  ← 后排：远程 pawn 站位
    └───────────┘
         │
    ┌────┴────┐
    │ 基地内部 │  ← 医院在侧翼（近入口易救治）
    └─────────┘
```

- 开阔区**不放柱子、岩块、树木** → 任何物体都可能给敌人掩体
- 沙袋+路障掩护效果相同（55%），交替布置无差别
- 开阔区露天（敌人黑暗，精度降低），我方射击位有屋顶
- 入口方向避开基地内部建筑物（流弹/火箭误伤）

### 近战堵门标准阵
```
敌人 →
  │
 墙墙│墙墙
 墙  │  墙       ← 墙厚 2 格，敌人一次只能进 1 格
 墙┌─门─┐墙
   │近│          ← 门外 1 格站 3 个近战 pawn（不站门内！）
   │近近│         ← 敌人被迫在门口 1v3
   │沙│           ← 后排沙袋掩体
   │远远│         ← 远程越过近战肩膀射击（无友伤）
   │远远│
```

- 3 个近战 pawn 站在**门外**，不在门内——站在门口内意味着敌人也进门
- 沙袋放在近战后方 2-3 格，远程躲在沙袋后射
- 手榴弹投掷点在**门口前方 2 格地面**，防误伤己方和墙
- 近战受伤后：后退→进门→关门→第二梯队顶替

## 材料选择
- **墙体**: 花岗岩 > 石灰岩 > 砂岩 > 大理石（耐久度排序）。大理石美观度高适合内部墙。
- **地板**: 无菌地板用于研究和医疗，钢地板用于生产区，木地板用于居住区
- **防御墙**: 花岗岩最佳（最高耐久），至少 2 格厚度

## 走廊设计
- 主走廊 3 格宽（避免拥堵）
- 室外通道 1 格即可，连接各建筑
- 重要走廊两端设门（防虫灾和空降袭击）

## 电力规划
- 电线埋在墙内美观且防火
- 关键设施用双路供电（主电网 + 电池备份）
- 电池室独立建造（防火隔离）
- 地热发电机用围墙保护

## 温度控制
- 每间房装通风口连接到走廊（走廊统一控温）
- 冷库双层墙壁 + 双层门（气闸设计）
- 加热器/空调各 1-2 台即可控制标准房间温度

## 防御布局

### 外墙系统
外墙围住整个基地，不留缺口
- **建造**: 使用 `designate_room` 画外围矩形，或用 `nine_grid_walled` 模板自带外墙

### 入口设计
- 主入口留 1 格开口，引导敌人走预设路线（不主动打门）
- 入口内侧设置陷阱走廊（陷阱**不能相邻**，至少间隔 1 格：陷阱→空地→陷阱）
- 陷阱用钢铁或木头制作，定期补充。陷阱不能相邻，交替排列
- 入口外侧放沙袋（`designate_build, thingDef_name=Sandbags`）减慢敌人

### 射击阵地
- 外墙拐角内侧放置沙袋（2-3 格），作为射击掩体
- 屋顶覆盖射击位（利用 `get_structure_layout` 确认有屋顶）
- 射击阵地间距 3-5 格，避免误伤

### 内部安全
- 重要设施（发电、库存、研究台）放在基地核心区
- 卧室区远离外墙，减少流弹伤害
- 电池室独立建造，与其他建筑隔开至少 1 格（防火）

### 建造工具
- `plan_add(pos_x, pos_y, end_x, end_y, color="蓝", label="厨房")` — 画规划草图标记
- `plan_list` — 查看所有规划标记，确认布局
- `plan_remove(pos_x, pos_y, end_x, end_y)` — 删除规划标记
- `designate_room(pos_x, pos_y, end_x, end_y, wall_stuff="BlocksGranite")` — 画围墙/房间
- `designate_build(thingDef_name="Sandbags", pos_x, pos_y)` — 放置沙袋
- `designate_build(thingDef_name="Door", pos_x, pos_y, stuff_defName="BlocksGranite")` — 装门
- `list_base_templates` → `apply_base_template(template_name="nine_grid_walled")` — 一体化方案

## 防御阵地模版

### 模版 1: 标准入口阵地（Killbox 入口）
- 13x13 围墙围出开放区，正面留 3 格宽开口
- 开口两侧各放 4 个沙袋掩体
- 后排：墙 + 沙袋交替排列的射击线（6 个射手位）

### 模版 2: 走廊阻击阵（1 格宽通道）
- 3xN 走廊，两侧花岗岩墙
- 近战位：出口格站 1 个穿重甲殖民者堵门
- 后排：走廊后方 3 格处排列远程射手
- 通道内陷阱交替排列：陷阱→空→陷阱（不能相邻）

### 模版 3: 墙角交叉火力（L 型阵地）
- 两个 5 格墙段成 90° 相交，内角开口
- 两翼各站 2 个远程射手，从侧翼集火被堵在转角的敌人
- 射击位前方放沙袋
- 注：殖民者不得站在炮塔爆炸范围旁

### 模版 4: 简易射击掩体
- 单排 5 格沙袋线（`designate_build(thingDef_name="Sandbags")`）
- 沙袋后 1 格处站远程射手（直接有掩体加成）
- 适合临时防御，无需围墙
