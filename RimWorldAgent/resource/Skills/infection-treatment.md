## 感染机制

感染是 RimWorld 最快的致命疾病，不治疗约** 1.2 天内死亡**。

### 哪些伤口会感染

| 伤口类型 | 感染几率 |
|---------|---------|
| 咬伤、烧伤、化学烧伤 | 30% |
| 冻伤 | 25% |
| 撕裂伤 | 20% |
| 其他流血伤口（割伤等） | 15% |
| 钝伤（淤青）、骨折、缺失部位 | 0%（不会感染） |

### 感染进程

感染一旦发生，就是**感染进度 vs 免疫进度**的竞速，先到 100% 者"获胜"：

- **不治疗**：感染每天 +0.84，约 1.19 天致死
- **100% 治疗质量**：感染每天 +0.31（大幅减缓）
- **100% 免疫增益速度**：免疫每天 +0.6441，约 1.55 天免疫

**关键**：100% IGS 时治疗质量需 ≥15% 才能存活。老年人、肝肾损伤者的 IGS 更低，需要更好治疗。

### 感染阶段

| 阶段 | 严重度 | 症状 |
|------|--------|------|
| 轻微 | 0-32% | 疼痛 +5% |
| 严重 | 33-77% | 疼痛 +8% |
| 极重 | 78-86% | 疼痛 +12%，意识 -5% |
| 极重 | 87-99% | 意识上限 10%（昏迷），疼痛 +85%，呼吸 -5% |
| 死亡 | 100% | 死亡 |

## AI 处理流程

### 预防（最佳策略）

1. **受伤后立即治疗**：用 `get_colonist_health` 查看伤口，用 `tend_now` 或等待医生自动治疗
2. **在干净环境治疗**：建造地板（Concrete/SteelTile）+ 定期清洁，无菌地板（SterileTile）最佳
3. **治疗质量影响**：治疗质量 0% → 感染率 ×85%；治疗质量 100% → 感染率 ×5%
4. **盘诺西林**：每 5 天服用一次，预防感染（通过 `get_colonist_health` 查看是否需要）

### 发现感染后的紧急处理

1. **立即查看免疫 vs 感染进度**：`get_colonist_health(colonist_id=<id>)` → 在 Health 标签中查看感染部位的"免疫"和"感染"百分比
2. **确保卧床休息**：用 `force_bed_rest(<id>)` 强制卧床（大幅加速免疫）
3. **确保医生治疗**：用 `get_work_priorities` 检查医生优先级，确保 Doctor 优先级 ≥ 2
4. **切换更好药物**：用 `get_colonists` 查看当前药物策略，考虑切换到工业药物或闪耀世界药物
5. **升级病房**：用 `get_device_info` 查看病床质量，考虑升级到医院床（HospitalBed）+ 生命体征监测仪（VitalsMonitor）

### 感染进度 > 80% 的极端情况

当感染超过 80% 且免疫进度落后时，必须采取极端措施：

1. **截肢（最可靠）**：用 `schedule_operation(colonist_id=<id>, operation="amputate_<body_part>")` 切除感染部位
   - 截肢在 87% 前有较高成功率，超过 90% 很危险，超过 97% 几乎来不及
   - 截肢无心情惩罚，失去四肢仍可安装义肢
2. **闪耀世界药物**：治疗效果远优于草药，可能反超免疫进度
3. **机械治愈液**：`get_device_info` 查找 HealerMechSerum
4. **冷却睡眠舱**：暂停感染进度，争取时间
5. **路西法菌**：最后手段，永久成瘾但可挽救生命

### 感染症状

感染后期（87%+）殖民者会**昏迷**（意识上限 10%），无法行走和进食，需要**喂食**（其他殖民者会自动执行）。确保有足够人手照料。

## MCP 工具速查

| 场景 | 工具 |
|------|------|
| 查看感染状态 | `get_colonist_health` |
| 强制治疗 | `tend_now(colonist_id)` |
| 强制卧床 | `force_bed_rest(colonist_id)` |
| 安排截肢 | `schedule_operation(colonist_id, "amputate_right_leg")` |
| 查看病床 | `get_device_info(thing_id)` |
| 建造医院床 | `designate_build(thing_def:"HospitalBed")` |
| 铺设无菌地板 | `designate_build(thing_def:"SterileTile")` |
